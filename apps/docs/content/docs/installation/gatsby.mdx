### Create project

Start by creating a new Gatsby project using create-gatsby:

```bash
npm init gatsby
```

### Configure your Gatsby project to use TypeScript and Tailwind CSS

You will be asked a few questions to configure your project:

```txt showLineNumbers
✔ What would you like to call your site?
· your-app-name
✔ What would you like to name the folder where your site will be created?
· your-app-name
✔ Will you be using JavaScript or TypeScript?
· TypeScript
✔ Will you be using a CMS?
· Choose whatever you want
✔ Would you like to install a styling system?
· Tailwind CSS
✔ Would you like to install additional features with other plugins?
· Choose whatever you want
✔ Shall we do this? (Y/n) · Yes
```

### Install the packages

Run the following command to install the packages:

With yarn:

```bash
yarn add @bigcomponents/core @bigcomponents/hooks tailwindcss-animate
```

With npm:

```bash
npm i @bigcomponents/core @bigcomponents/hooks tailwindcss-animate
```

With pnpm

```bash
pnpm add @bigcomponents/core @bigcomponents/hooks tailwindcss-animate
```
