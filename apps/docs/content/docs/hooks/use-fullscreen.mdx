## Usage

`use-fullscreen` allows to enter/exit fullscreen for given element using the [Fullscreen API](https://developer.mozilla.org/en-US/docs/Web/API/Fullscreen_API). By default, if you don't provide `ref`, hook will target `document.documentElement`:

<ComponentPreview name="use-fullscreen-demo" />

## Custom root element

Hook returns an optional ref function that can be passed to an element to act as root. Be sure to follow best practices to not [confuse or trap the end user:](https://developer.mozilla.org/en-US/docs/Web/API/Fullscreen_API/Guide#things_your_users_want_to_know)

## Definition

```tsx
function useFullscreen<T extends HTMLElement = any>(): {
  readonly ref: (element: T | null) => void;
  readonly toggle: () => Promise<void>;
  readonly fullscreen: boolean;
};
```
